Основы HTML

HTML (HyperText Markup Language) - это стандартный язык разметки для создания веб-страниц. Он определяет структуру и содержание веб-страницы.

Базовая структура HTML-документа

Каждый HTML-документ начинается с декларации типа документа <!DOCTYPE html>, за которой следует элемент <html>.

Пример базовой структуры HTML 
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Мой первый сайт</title>
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это мой первый HTML-документ.</p>
</body>
</html>

Основные разделы документа

<head>

Содержит метаинформацию о документе: заголовок, ссылки на стили и скрипты, кодировку и другие метаданные.

<body>

Содержит видимое содержимое страницы: тексты, изображения, ссылки, таблицы, формы и другие элементы.

Основные HTML-элементы

HTML состоит из элементов, которые определяют структуру и содержание веб-страницы.

Заголовки

Заголовки от <h1> до <h6> определяют важность текста. <h1> - самый важный, <h6> - наименее важный.

Пример 
<h1>Главный заголовок</h1>
<h2>Подзаголовок</h2>
<h3>Менее важный заголовок</h3>

Абзацы

Элемент <p> определяет абзац текста. Браузеры автоматически добавляют отступы до и после каждого абзаца.

Пример 
<p>Это первый абзац.</p>
<p>Это второй абзац.</p>

Ссылки

Элемент <a> создает гиперссылки. Атрибут href указывает URL-адрес страницы, на которую ведет ссылка.

Пример 
<a href="https://example.com">Посетите Example.com</a>

Изображения

Элемент <img> вставляет изображение на страницу. Атрибут src указывает путь к изображению, а alt - альтернативный текст.

Пример 
<img src="image.jpg" alt="Описание изображения">

HTML-атрибуты

Атрибуты предоставляют дополнительную информацию об элементах и всегда указываются в начальном теге.

Основные атрибуты

id, class, style, title - универсальные атрибуты, которые можно использовать с большинством элементов.

Пример 
<p id="intro" class="important" title="Введение">
    Это важный текст.
</p>

Атрибуты ссылок

href, target, download, rel - специальные атрибуты для элемента <a>.

Пример 
<a href="page.html" target="_blank" rel="noopener">
    Открыть в новой вкладке
</a>

Атрибуты изображений

src, alt, width, height, loading - специальные атрибуты для элемента <img>.

Пример 
<img src="photo.jpg" alt="Пейзаж" 
     width="800" height="600" loading="lazy">

HTML5 Элементы

HTML5 принес множество новых семантических элементов, которые улучшают структуру веб-страницы.

<header>

Определяет шапку документа или раздела. Обычно содержит заголовки, логотип и навигацию.

<footer>

Определяет подвал документа или раздела. Обычно содержит информацию об авторских правах, контакты и ссылки.

<nav>

Определяет блок навигационных ссылок. Используется для основной навигации по сайту.

<section>

Определяет тематический раздел в документе. Обычно содержит заголовок и связанное содержимое.

<article>

Определяет независимый, самодостаточный контент, который может быть распространен отдельно от сайта.

<aside>

Определяет контент, косвенно связанный с основным содержимым (боковые панели, цитаты и т.д.).

Пример структуры страницы с HTML5 элементами 
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Современная структура HTML5</title>
</head>
<body>
    <header>
        <h1>Заголовок сайта</h1>
        <nav>
            <ul>
                <li><a href="#">Главная</a></li>
                <li><a href="#">О нас</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <article>
            <h2>Заголовок статьи</h2>
            <p>Содержание статьи...</p>
        </article>
        
        <aside>
            <h3>Дополнительная информация</h3>
            <p>Рекламный блок или другая информация...</p>
        </aside>
    </main>
    
    <footer>
        <p>© 2023 Мой сайт</p>
    </footer>
</body>
</html>
